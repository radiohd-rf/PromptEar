# Чек-лист: Видео

## 🛠️ Чем проверять

**ffmpeg (встроен в приложение) — сделать видео без звука:**
ffmpeg -i normal.mp4 -an -c copy silent.mp4
Потом загрузи это silent.mp4 — должна быть ошибка "нет аудиодорожки"

**ffprobe — проверить что внутри видео:**
ffprobe video.mp4
Смотри секцию Audio: если streams[0] только Video — дорожки нет. Если Audio есть — должно распознаться.

**Как сделать битый файл:**
просто открой любой mp4 в блокноте, сотри пару символов, сохрани. Или через PowerShell:
echo "битый" > broken.mp4

**DevTools (F12 через браузер):**
Network → EventStream — при загрузке битого видео должно прийти событие error с текстом "Не удалось прочитать видео" или "В видео ... нет аудиодорожки".

**Обычные видео:**
просто накидай пару mp4, avi, mkv, mov, webm — все должны обработаться, результат в output/.

## Техники: EP (форматы), State Transition (извлечение), негатив

### Эквивалентные классы — форматы

- **Валидные:** MP4, AVI, MKV, MOV, WEBM, WMV, M4V, FLV, TS
- [ ] Каждый формат: загрузить → аудио извлекается → транскрибируется
- [ ] Видео распознаётся как видео (расширение из VIDEO_EXTENSIONS)

### Извлечение аудио

- [ ] ffmpeg извлекает аудио: pcm_s16le, 16000 Hz, mono
- [ ] Извлечённый WAV — временный, удаляется после обработки (Cleanup)
- [ ] Оригинальный видеофайл НЕ удаляется до завершения обработки
- [ ] После обработки оригинальный видеофайл удаляется

### Влияние на output

- [ ] Имя выходного файла — от оригинального видео (recording.mp4 → recording.docx)
- [ ] DOCX/TXT — содержимое как для аудио

### Негативные сценарии

- [ ] **Видео без аудиодорожки:** ошибка «В видео ... нет аудиодорожки» в логе
- [ ] **Битое видео:** ошибка «Не удалось прочитать видео»
- [ ] **Видео с очень длинным именем:** извлечение аудио работает без ошибки
- [ ] **Видео 4K/большой битрейт:** извлечение аудио не падает
- [ ] **Видео 0 байт:** ошибка ffmpeg с fallback

### Mix файлов

- [ ] Загрузить 2 аудио + 2 видео → все 4 обрабатываются последовательно
- [ ] Видео + аудио вместе — pipeline корректно различает типы
