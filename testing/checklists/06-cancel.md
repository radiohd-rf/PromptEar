# Чек-лист: Отмена обработки

## 🛠️ Чем проверять

**DevTools (F12 через браузер):**
Network:
- найди POST /api/cancel после нажатия «Отмена»
- статус должен быть 200 (или 204). Если 404 — что-то с task_id
Console:
- после отмены должно быть событие cancelled в SSE. Если ошибка — красная строка в console

**Проводник:**
- открой папку output/
- после отмены проверь — уже обработанные файлы сохранились?

**Process Explorer (скачать с learn.microsoft.com/sysinternals):**
- запусти procexp.exe
- найди python.exe (Ctrl+F по имени)
- нажми «Отмена» в приложении — смотри что python.exe исчез, ffmpeg тоже
- если завис процесс — баг

**Если Process Explorer нет:**
можно через Task Manager (Ctrl+Shift+Esc) глянуть — там процессы python и ffmpeg должны пропадать после отмены.

## Техники: State Transition (отмена на каждом шаге), EP, негатив

### State Transition — отмена на каждом шаге

Загрузить 1 большой файл, нажать «Обработать», затем «Отмена»:

- [ ] **Отмена на шаге 1 (DetectPreprocess):** ffmpeg убивается, pipeline сразу завершается
- [ ] **Отмена на шаге 2 (Transcribe):** Whisper прерывается между сегментами
- [ ] **Отмена на шаге 3 (Enhance):** Qwen прерывается между чанками/проходами
- [ ] **Отмена на шаге 4 (Save):** файл не сохраняется?
- [ ] **Отмена на шаге 5 (Cleanup):** cleanup всё равно выполняется

### Частичные результаты

- [ ] Обработать 5 файлов, отменить на середине — уже завершённые файлы сохранены в output/
- [ ] Отменить, снова запустить — предыдущие файлы НЕ перезаписываются

### UI при отмене

- [ ] Кнопка «Отмена» появляется при старте
- [ ] При нажатии «Отмена» — отправляется POST /api/cancel/{task_id}
- [ ] Лог: «⛔ Обработка отменена пользователем»
- [ ] Спинсер скрывается
- [ ] Кнопка «Обработать» восстанавливается
- [ ] Кнопка «Отмена» скрывается

### Негативные сценарии

- [ ] **Двойная отмена:** нажать «Отмена» дважды — второй запрос не вызывает ошибку
- [ ] **Отмена после завершения:** нажать «Отмена» после Done — ничего не происходит
- [ ] **Отмена без активной задачи:** POST /api/cancel/nonexistent_id — 404
- [ ] **Закрыть окно во время обработки:** задача прерывается, временные файлы?
