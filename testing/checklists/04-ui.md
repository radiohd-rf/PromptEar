# Чек-лист: Пользовательский интерфейс

## Техники: State Transition, EP

### Тема

- [ ] По умолчанию — тёмная тема
- [ ] Нажать ☀️ — переключение на светлую тему
- [ ] Нажать 🌙 — переключение обратно на тёмную
- [ ] При переключении темы меняются: фон, текст, кнопки, дроп-зона, лог, модалка, скроллбар, футер
- [ ] Тема сохраняется после перезагрузки страницы? (нет persistence)
- [ ] Состояние кнопки темы: ☀️ на тёмной теме, 🌙 на светлой

### Модалка Help («?»)

- **State Transition:**
  - [ ] Закрыта → нажать «?» → открыта
  - [ ] Открыта → нажать ✕ → закрыта
  - [ ] Открыта → нажать Escape → закрыта
  - [ ] Открыта → клик вне модалки (на overlay) → закрыта
  - [ ] Открыта → клик внутри модалки → НЕ закрывается
- [ ] Содержимое модалки: инструкция по use-кейсам, drag-drop, контекст, форматы, pipeline, отмена, папка output, статус-бар
- [ ] Модалка скроллится при длинном содержимом
- [ ] Модалка центрирована на экране
- [ ] Максимальная ширина ~520px

### Спиннер

- [ ] Спиннер скрыт при загрузке страницы
- [ ] Спиннер появляется при нажатии «Обработать»
- [ ] Спиннер скрывается при завершении обработки
- [ ] Спиннер скрывается при отмене
- [ ] Спиннер анимирован (вращается)
- [ ] Цвет спиннера соответствует теме

### Лог-панель

- [ ] Лог скрыт, когда нет обработки
- [ ] Лог появляется при старте обработки
- [ ] Сообщения в логе: ✅, ⚠, ❌, ⛔, 📁
- [ ] Лог автоматически скроллится вниз при новых сообщениях
- [ ] Лог очищается при новом запуске
- [ ] Моноширинный шрифт (Consolas/Courier New)
- [ ] Тёмный скроллбар в логе

### Кнопки: «Обработать»

- [ ] Видна, когда нет обработки
- [ ] Скрыта во время обработки
- [ ] Disabled, когда нет файлов в списке
- [ ] Enabled, когда есть файлы
- [ ] Нажатие без файлов — ничего не происходит

### Кнопки: «Отмена»

- [ ] Скрыта, когда нет обработки
- [ ] Видна только во время обработки
- [ ] Нажатие отправляет POST /api/cancel/{task_id}
- [ ] После отмены обе кнопки восстанавливаются

### Скроллбар

- [ ] Кастомный скроллбар (8px, тёмный в тёмной теме, светлый в светлой)
- [ ] Скроллбар в лог-панели
- [ ] Скроллбар в списке файлов
- [ ] Скроллбар в модалке
- [ ] На body скролл отсутствует (overflow: hidden)
