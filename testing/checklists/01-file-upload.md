# Чек-лист: Загрузка файлов

## 🛠️ Чем проверять

**Где взять тестовые файлы:**
- свои mp3 — просто накидай из музыки/записей
- пустой файл — открой блокнот, сохрани пустой test.mp3
- длинное имя — в PowerShell: New-Item "A"*255 + ".mp3"
- unicode в имени — переименуй файл в "тест(экспорт).mp3" прямо в проводнике
- битый файл — открой mp3 в блокноте, допиши пару символов, сохрани

**Как открыть DevTools:**
приложение запускается в PyWebView — F12 там не работает. Просто открой браузер (Chrome) и перейди на http://127.0.0.1:5000 (или какой порт выдало при первом запуске — напишет в консоли). Тот же интерфейс, но можно жать F12.

**Что смотреть в DevTools:**
- Console — красные ошибки при драг-н-дропе
- Network — найти POST /api/files, открыть, смотреть Form Data (имена файлов, формат)
- Elements — как меняется счётчик "Добавлено файлов: N"

**В проводнике:**
просто таскай файлы в окно приложения, жми ✕, смотри что список обновляется. Проверь что файлы .txt .pdf не добавляются.

## Техники: EP, BVA, State Transition, негатив

### Drag-and-drop

- [ ] Перетащить 1 аудиофайл (MP3) — добавляется в список
- [ ] Перетащить 1 видеофайл (MP4) — добавляется в список
- [ ] Перетащить файл неподдерживаемого формата (.txt, .pdf, .exe) — игнорируется
- [ ] Перетащить пустой файл — игнорируется / ошибка
- [ ] Перетащить файл с длинным именем (>255 символов) — добавляется без ошибки
- [ ] Перетащить файл с Unicode в имени — добавляется без ошибки
- [ ] Перетащить файл с пробелами и спецсимволами в имени
- [ ] Drop за пределами drop-zone (на фон) — файл НЕ открывается в браузере

### Диалог выбора файлов

- [ ] Нажать «Выбрать файлы» — открывается системный диалог
- [ ] Выбрать 0 файлов (отмена в диалоге) — ничего не меняется
- [ ] Выбрать 1 файл через диалог — добавляется в список
- [ ] Выбрать несколько файлов через Ctrl/Shift — все добавляются
- [ ] Выбрать все поддерживаемые форматы разом (mix аудио + видео)

### Множественная загрузка

- **EP:** 1, 5, 10, 50 файлов — все добавляются
- **BVA:** 0 файлов, 1 файл, 100 файлов — граничные значения
- [ ] Добавить 100+ файлов — UI не зависает, список прокручивается

### Дубликаты

- [ ] Загрузить один и тот же файл дважды (одинаковое имя + размер) — дубликат НЕ добавляется
- [ ] Загрузить файлы с одинаковым именем, но разным размером — оба добавляются
- [ ] Загрузить 2 одинаковых файла, потом удалить один, добавить снова — добавляется

### Управление списком

- [ ] После добавления файлов счётчик показывает: «Добавлено файлов: N»
- [ ] Нажать ✕ на файле — файл удаляется из списка, счётчик уменьшается
- [ ] Удалить все файлы — список скрывается, счётчик пропадает
- [ ] Список файлов не отображается, когда файлов нет

### Негативные сценарии

- [ ] Перетащить папку (не файл) — игнорируется
- [ ] Перетащить ярлык (.lnk) — игнорируется или обрабатывается как целевой файл
- [ ] Перетащить файл с расширением .MP3 (верхний регистр) — поддерживается?
- [ ] Перетащить файл размером 0 байт — ошибка или игнорирование
- [ ] Загрузить файлы разных форматов, затем нажать «Обработать» — все обрабатываются
